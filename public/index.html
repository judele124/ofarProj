Toys API

HTTP Method Route Description
GET toys Returns a list of toys according to the query parameters.
Query Parameter Description Default
limit The number of objects to return in the response. 10
page Page number for pagination. 0
sort Sort order from big to small. price
reverse Sort order from small to big. From big to small
s Search by name and info. None
cat Search by category None
HTTP Method Route Description Parameters / Request Body
GET /toys/single/:id Returns a single toy by its ID.

id: the ID of the toy to retrieve.

/toys/single/64f46142409942f6717d7858
POST /toys Adds a new toy. Token is required!

Request Body: Toy data to be added in the following format.

price: 1-99999

img_url is allowed and not required

 
{
    name: String,
    info: String,
    category: String,
    price: Number,
    img_url: String 
}
PUT /toys/:id Updates an existing toy. Token is required!

id: the ID of the toy to update.

Request Body: Toy data to be updated in the following fornat

price: 1-999999

img_url is allowed and not required

 
 {
     name: String,
     info: String,
     category: String,
     price: Number,
     img_url: String 
 }
DELETE /toys/:id Deletes a toy by its ID. Token is required!

id: the ID of the toy to delete.
